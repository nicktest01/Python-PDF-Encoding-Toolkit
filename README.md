# Python-PDF-Encoding-Toolkit

## 🌟 功能特點
1. **編碼轉換器**：自動將 Big5 (繁體中文) 檔案轉換為 UTF-8，徹底解決亂碼問題。
2. **PDF 生成器**：將 TXT 內容轉換為 PDF，並完美支援中文顯示（不再需要手動處理複雜的 JAR 包）。
3. **網頁預覽器**：建立一個 Flask 微網頁，提供按鈕讓使用者直接在瀏覽器開啟 PDF。
